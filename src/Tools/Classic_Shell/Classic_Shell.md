## Tools

### Classic Shell / Open Shell Menu

Open-Shell ist ein quelloffenes Dienstprogramm für das Betriebssystem Windows. Es ist eine Weiterentwicklung von Classic Shell, welches ursprünglich von Ivo Beltchev entwickelt und 2017 eingestellt wurde. Open-Shell bietet unter anderem das klassische Startmenü für Windows 7, 8, 8.1 und 10 an.
