## Tools

### Beyond Compare

Beyond Compare ist ein Datenvergleichsprogramm. Neben dem Vergleich von Dateien kann das Programm auch gesamte Verzeichnisse, FTP- und SFTP- Verzeichnissen, Dropbox, Amazon S3 sowie Archive gegenüberstellen. Die Software ist für Windows, Mac OS und Linux verfügbar.
